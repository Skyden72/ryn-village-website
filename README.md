# Ryn Village Website

Premium retirement estate website with modern design, parallax effects, and elegant animations.

## Project Structure

This repository contains **two versions** of the Ryn Village website:

### 1. Next.js Version (`/web`)
Modern React-based landing page with:
- Framer Motion animations
- Tailwind CSS styling
- React Three Fiber ready (for 3D elements)
- Optimized for Vercel/Netlify deployment

**Run locally:**
```bash
cd web
npm install
npm run dev
```

### 2. WordPress Theme (`/Rynvillage Wordpress`)
Traditional WordPress theme with:
- PHP templates
- Vanilla CSS + JavaScript
- Custom Post Type for Units
- Theme Customizer integration

**Install:**
1. Copy `Rynvillage Wordpress` folder to `/wp-content/themes/`
2. Activate in WordPress Admin → Appearance → Themes

## Features

- ✅ Hero section with parallax background
- ✅ Animated amenities grid
- ✅ Unit showcase carousel
- ✅ Life Rights information section
